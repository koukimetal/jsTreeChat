# jsTreeChat
Chat using jsTree

##How to deploy
* Open `config.php` and change following parameters
  * MASTER_PASS
  * DATABASE_USER
  * DATABASE_PASS
* Remove `.gitignore` and `README.md`
* Send remains
* Make permission of `core.php` 755
